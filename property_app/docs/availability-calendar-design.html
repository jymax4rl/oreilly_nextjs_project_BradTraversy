2. Date rules

Stay: checkIn 2026-06-10, checkOut 2026-06-15
Blocked nights: 10, 11, 12, 13, 14 (checkOut day = departure, not booked)

Storage: YYYY-MM-DD (UTC date-only)
checkIn inclusive · checkOut exclusive

4. API flow — GET availability
